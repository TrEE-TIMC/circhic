circhic.datasets.load_ccrescentus¶

circhic.datasets.load_ccrescentus()¶

Loads C. crescentus contact counts.

Returns
dictionary :
a dictionary containing:

  • counts: an (n, n) ndarray corresponding to the raw contact counts for C. crescentus

  • nbins: (l, ) ndarray containing the number of bins of all chromosomes.
